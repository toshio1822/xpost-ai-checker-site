## 📘 このシリーズで伝えたいこと

1. **削除より「記録」を優先すること。**  
2. **焦らず冷静に、感情的な反応を避けること。**  
3. **正しい通報と相談ルートを知ること。**  
4. **法律知識を持ち、被害者で終わらないこと。**
