## 📘 このシリーズで伝えたいこと

1. **通報は「ゴール」ではなく「スタート」。**  
2. **AI証拠化で、被害の事実を正確に残す。**  
3. **専門機関・公的機関への相談は、ためらわないこと。**  
4. **感情的な投稿は、逆に不利になるリスクがある。**
